# AI Research Assistant

An AI-powered research assistant that automatically searches, summarizes, and generates research reports from multiple articles using a local Large Language Model.

---

## 📌 Project Overview

The AI Research Assistant is designed to help students, researchers, and professionals quickly understand complex topics. Instead of manually reading multiple articles, the system collects information, summarizes it using AI, and generates a final research report.

The project runs locally using large language models through Ollama.

---

## 🚀 Features

- Automatic research topic analysis
- Article retrieval and processing
- AI-powered text summarization
- Automatic report generation
- Runs completely on a local machine
- Fast research assistance

---

## 🧠 Technologies Used

- Python
- LangChain
- Ollama
- Mistral / Llama3 LLM
- Requests Library
- VS Code
